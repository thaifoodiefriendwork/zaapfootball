โครงสร้าง Ads

ประเภทรายละเอียด
1) Awareness 🚀 เป้าหมาย: สร้างกระแส “ดูบอล = ต้องมีแซ่บ”
คอนเทนต์: No Popcorn YES เห็ดถัง, ดูบอลกับเพื่อน, ฟีลห้องดูบอล, Reaction บอล ฯลฯ
KPI: Reach, Views, Engagement
ทดสอบ: Hook 3–5 แบบ, 6–15s, Caption 2 เวอร์ชัน
2) Conversion 🛒 เป้าหมาย: ปิดการขาย
สินค้าหลัก: SET A (539.-), น้ำพริก 40g, เซ็ตดูบอล SET B/C/D
KPI: ROI, CPA, Conversion
เสริม: รีมาร์เก็ตฯ วิดีโอ/มีส่วนร่วม/เข้าชม, ใส่ UGC 5–8s
3) Live Ads 🔴 เป้าหมาย: ดึงคนเข้า Live
ช่วงยิง: ก่อนเตะ, พักครึ่ง, แมตช์ใหญ่, หลังยิงประตู
KPI: คนเข้าไลฟ์, CTR, Conversion Live
ทริก: ปุ่ม “ดูสดรับโค้ด”, “ยิงประตูแจกโค้ด”, ใส่เวลาคิกออฟบนภาพ
